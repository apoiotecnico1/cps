Guia rápido

Como esta base ajuda novatos e veteranos

  1. Use “Nova Nota” para registrar padrões, decisões e lições aprendidas.
  2. Classifique por Tipo (Técnico, Procedimento, Incidente, etc.), Status e Tags.
  3. Anexe PDFs/fotos e cole links do SharePoint para referência.
  4. Pesquise pelo campo global (título, tags e conteúdo).
  5. Exportar JSON para backup/compartilhar e Importar JSON para atualizar.

Templates úteis

  • Procedimento padrão: objetivo → EPI → materiais → passos → validação.
  • Incidente: descrição → impacto → causa raiz → ação corretiva → preventiva.
  • Checklist preventiva: itens → critérios → medições → aceite.
  • FAQ: pergunta → resposta breve → referência.
